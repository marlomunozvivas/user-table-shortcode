A simple shortcode plugin that displays registered users in a responsive table with filtering, searching, and export to Excel functionality.

== Description ==
User Table Shortcode is a lightweight plugin that helps WordPress administrators and developers display a responsive, searchable, and filterable table of registered users on any page using a simple shortcode.

Features include:
* Responsive table layout.
* Filter by user role.
* Export user data to Excel (CSV format).
* Easy integration with a shortcode: `[user_table]`.

== Installation ==
1. Upload the `user-table-shortcode` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Add the shortcode `[user_table]` to any page or post where you want the user table to appear.

== Frequently Asked Questions ==
= How do I use the plugin? =
Simply add the `[user_table]` shortcode to any page or post.

= Can I customize the styles? =
You can override the styles using custom CSS in your theme.
